# Workshop_IA_Erasme_APP

Prerequisite:
  - Android SDK
  - ADB Driver
  - Nodejs & ionic

Installation:
 - git clone https://github.com/LucasALLOIN/Workshop_IA_Erasme_APP
 - cd Workshop_IA_Erasme_APP
 - npm install
 - /!\ Plug smart device
 - ionic cordova platform add android
 - ionic cordova run android --device
